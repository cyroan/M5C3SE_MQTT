# M5C3SE_MQTT 开发纪录

本项目是基于 **M5Stack CoreS3** 开发的 MQTT 监控与 OTA 更新系统，支持双网切换及离线历史回顾。

## 版本演进 (Development Log)

### v1.1.9 (Current)
- **自動運行邏輯優化**：修正了 30 秒自動運行時未執行連線動作的 Bug。現在會自動載入儲存的 WiFi/LAN 設定並嘗試連線。
- **MQTT 穩定性**：在連線 MQTT 前加入網路狀態檢查，避免在網路未就緒時嘗試連線導致系統不穩定或重啟。
- **歷史導航修復**：修正監控畫面 UP 按鈕無法正確回溯歷史紀錄的問題。
- **版本同步**：更新 `version.h` 為 1.1.9。

### v1.1.8
- **狀態轉換修復**：修正了 `loop()` 中 30 秒自動運行邏輯可能導致已進入監控模式後又跳回啟動頁面的 Bug。
- **觸控防誤觸**：縮小了 `STATE_RUNNING` 模式下返回首頁（HOME）的觸控判定區域，降低因螢幕下方誤觸導致的意外退出。

### v1.1.7
- **键盘 UI 优化**：所有功能键（BS, KB, HOME, OK, SPC）字体加大至 Size 2。
- **导航修复**：重更名为 `KB` 键，修复了 `HOME` 按钮返回失效的 Bug。
- **流程加固**：优化了各设置步骤间的防抖与画面重绘。

### v1.1.6
- **键盘架构调整**：将 `BACK` 功能拆分为分页切换（KB）与全局返回（HOME）。
- **RTC 增强**：Value+ / Value- 按钮字型加大，响应更稳健。

### v1.1.5
- **自动运行**：恢复 30 秒无操作自动进入监控模式的功能。
- **重绘修复**：解决从设置界面退出时主界面的乱码残留问题。

### v1.1.4
- **MQTT 引导化**：将 MQTT 设置改为 Server -> Port -> Sub -> Pub 的分步引导流程。
- **UI 紧凑化**：监控界面 Topic 直接显示，时间戳字型加大。

### v1.1.0 - v1.1.3
- **功能集成**：加入 RTC 手动校时（支持 +/-）。
- **排版优化**：监控界面 Left Justify 对齐，增加 SD 卡状态指示灯（绿/红圆点）。

### v1.0.6 - v1.0.9
- **滚动支持**：实现 WiFi 列表的翻页滚动功能。
- **历史记录**：引入最近 10 笔 MQTT 消息的循环缓存与 Review 功能。

### v1.0.0 - v1.0.5
- **核心功能**：支持 WiFi 与 LAN Module 13.2 双网选择。
- **OTA 支持**：通过 GitHub (HTTPS) 实现无线固件更新（限 WiFi 模式）。
- **项目更名**：由 `M5STACK_OTA_Test` 正式更名为 `M5C3SE_MQTT`。
